# foodlog

Web app FoodLog — Vite + React + TypeScript + Tailwind CSS.

## Chạy local

```bash
npm install
npm run dev
```

Mở http://localhost:5173

## Build

```bash
npm run build
npm run preview
```

## Deploy

Push lên nhánh `master` → GitHub Actions tự build và deploy lên GitHub Pages.

Site: https://giapsoft.github.io/foodlog/
